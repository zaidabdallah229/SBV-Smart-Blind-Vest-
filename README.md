# SBV (Smart Blind Vest) - السترة الذكية للمكفوفين

نظام **السترة الذكية (BlindVest)** هو مشروع متكامل يهدف إلى مساعدة المكفوفين وضعاف البصر على التنقل بأمان واستقلالية باستخدام تقنيات الذكاء الاصطناعي (رؤية الحاسب) وإنترنت الأشياء (IoT). يتكون النظام من تطبيق بايثون (معالجة الصور والذكاء الاصطناعي) وكود أردوينو (للتحكم في المستشعرات والمحركات).

---

## ✨ المميزات الرئيسية

1. **التعرف على الأشياء بالذكاء الاصطناعي (YOLOv8):** 
   - يتعرف على العوائق المحيطة (سيارات، أشخاص، أبواب، كراسي، إلخ).
   - يميز حالة إشارات المرور (خضراء للانطلاق، وحمراء للوقوف).
2. **التوجيه الصوتي باللغة العربية:**
   - يخبر المستخدم بنوع العائق وموقعه (يمين، يسار، أو في المنتصف) عبر محرك `edge_tts`.
3. **التوجيه الحسي (محركات الاهتزاز):**
   - 3 محركات اهتزاز (يمين، وسط، يسار) متصلة بلوحة ESP32، تهتز بناءً على موقع العائق لتنبيه المستخدم حسياً.
4. **زر الطوارئ وتتبع الموقع (GPS):**
   - زر طوارئ مدمج في السترة يرسل تنبيهاً فورياً إلى لوحة التحكم مع موقع المستخدم الحالي عبر مستشعر GPS.
5. **لوحة تحكم للمراقبين (Web Dashboard):**
   - واجهة تحكم متطورة مصممة بألوان مريحة للعين، تتيح للمرافقين متابعة حالة المستخدم، استلام تنبيهات الطوارئ، ورؤية الموقع الجغرافي.

---

## 🛠️ التقنيات المستخدمة

### قسم الذكاء الاصطناعي (Python)
- **Python / Flask / Flask-SocketIO:** لبناء خادم الويب وربطه بالواجهة الأمامية في الوقت الفعلي.
- **OpenCV & YOLOv8:** لمعالجة الفيديو المباشر والتعرف الدقيق على الأشياء وإشارات المرور.
- **Edge TTS & Pygame:** لتوليد نطق صوتي عربي سريع ومستقر خالي من التقطيع.

### قسم الأجهزة (Arduino / ESP32)
- **ESP32 Microcontroller:** عقل السترة المتصل بشبكة الواي فاي للاتصال بالخادم.
- **TinyGPS++:** لقراءة إحداثيات الموقع (خطوط الطول والعرض).
- **PWM Motors:** محركات اهتزاز تعمل بتقنية PWM لتحديد قوة الاهتزاز بناءً على القرب.

## 📂 هيكل المشروع

يتكون المستودع من قسمين رئيسيين:

1. **`PythonApp/`**: يحتوي على كود الذكاء الاصطناعي (`robocraft.py`)، نموذج YOLO (`yolov8n.pt`)، وملفات الويب (`templates/index.html`).
2. **`Arduino/`**: يحتوي على كود التحكم بالعتاد (`Blind_final.ino`) الجاهز للرفع على شريحة ESP32.

## 🚀 طريقة التشغيل وإعداد النظام

### 1️⃣ إعداد قسم بايثون (السيرفر والكاميرا)
1. تأكد من تثبيت بايثون على جهازك.
2. قم بتثبيت المكتبات المطلوبة عبر الأمر التالي:
   ```bash
   pip install flask flask-socketio opencv-python ultralytics edge-tts pygame arabic_reshaper python-bidi requests
   ```
3. (اختياري) للحصول على أداء أفضل في الكاميرا، قم بتفعيل تسريع كرت الشاشة (CUDA) لمكتبة PyTorch.
4. قم بتشغيل الملف الرئيسي:
   ```bash
   cd PythonApp
   python robocraft.py
   ```
5. افتح المتصفح على `http://localhost:5000` للدخول إلى لوحة التحكم (كلمة المرور الافتراضية للجهاز `BLIND-001` هي `1234`).

### 2️⃣ إعداد قسم الأردوينو (ESP32)
1. افتح برنامج **Arduino IDE**.
2. أضف روابط دعم لوحة ESP32 وقم بتثبيتها من Boards Manager.
3. قم بتثبيت مكتبة `TinyGPS++` من Library Manager.
4. افتح ملف `Arduino/Blind_final.ino`.
5. قم بتعديل بيانات شبكة الواي فاي (`ssid` و `password`) ورقم الآي بي (`serverPath`) ليتطابق مع آي بي جهاز الكمبيوتر الذي يعمل عليه كود البايثون.
6. ارفع الكود على لوحة ESP32.
